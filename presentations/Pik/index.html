An introduction to PIK7

HTML5 presentation framework

Press → or tap right to learn more

Features

  • Dead simple – nothing but HTML, CSS and jQuery
  • Works in every Browser that's not IE 8 or older
  • Multiple slide views and presenter mode
  • Good performance in modern browsers even with hundreds of slides

Other features

  • Reasonably printable (can be used for PDF export too)
  • Rapid navigation between slide sets

Downsides

  • Requires a modern web browser and web server *
  • No fancyful animations **
  • Shares all drawbacks common to most HTML slides (e.g. no GUI to create presentations)

* Pik7 actually has it's own server, but it in turn requires Node.js
** Slide transitions are possible but require some advanced css wizzardy

Getting started

  1. Download Pik7 from Github or git clone git://github.com/SirPepe/Pik7.git
  2. Place the files in a directory on your web server
  3. Open Pik7 in a browser, click the "Browse" link to navigate through the slide sets

Controls

  • Use the arrow keys and PGUP/PGDN to navigate through the slides
  • The F5 and ESC keys hide the presentation
  • On mobile devices you can tap left and right to navigate
  • The control panel can be found on the bottom right

Building presentations

<!-- /presentations/Foo/bar.html -->
<html id="PikSlides">
<script src="../../core/pik7.js"></script>

<div class="pikSlide">First Slide</div>
<div class="pikSlide">Second Slide</div>

Use presentations/Template/index.html as a starting point.

Styling presentations

  • Change the theme by setting the data-theme attribute on the <script> tag that loads pik7.js
  • Example: <script data-theme="themes/template" src="../../core/pik7.js"></script>
  • prefix-free is included by default

Notes

  • Notes can be added using an element with the class pikNotes for each slide
  • Notes are invisible in the presentation, but are shown in the presenter view

Use all the tags you like!

  • No limitations! Use links, headlines and everything else!
  • Notes won't be printed

Printing presentations

  • Use the print link in the control panel to print your presentation
  • A very basic print stylesheet is included; use your own style file to enhance it to your liking
  • Hide non-printable elements by adding the class pikNoprint

Programming presentations

  • pikActivate fires on a slide when it becomes the current slide, pikDeactivate fires when the slide ist navigated away from
  • pikSlide fires on window each time slides change, passing the current slide element and it's number
  • pikHide and pikShow fire on window when the hidden state changes

Simple demo

$(document).ready(function(){
  $('#EventsDemo').bind({
    pikActivate: function(){
      $(this).find('p').addClass('awesome');
    },
    pikDeactivate: function(){
      $(this).find('p').removeClass('awesome');
    }
  });
});

Awesome!

Special css classes

  • pikNoprint prevents an element from being printed
  • pikNotes

Slide transitions

  • Slide visibility is completly managed by css; all slides except the one with the class pik-current are invisible by default
  • By changing the behavior of this class and the classes pik-next and pik-prev it is possible to create slide transitions
